About

This server is a casual server for the giga-est of chads focusing on mostly the Create mod. The idea is that it puts less stress on the players that are new to modded, while giving a refreshed Minecraft experience.

We also have a Lightly Modded Valheim server, which can be viewed here.

Minimum Requirements

Minecraft Java Edition 1.20.1 on any supported PC. Bedrock edition is not supported.

This server is not vanilla conpatible. Please use the modpack linked below.

Using Minecraft With Mods

Fear not, for it is not hard with the right launcher.

There a few great launchers that make it easy to add mods, such as Prism Launcher. You will need to install Java seperately for either of these launchers. Follow the guided setup for the Java installation and your launcher of choice.

Please ask in the Discord if you need help with setting up the modded launchers.
