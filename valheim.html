About

This server is a casual server for the giga-est of chads focusing on mostly QoL mods. The idea is that it puts less stress on the players that are new to modded, while giving older players a refreshed Valheim experience.


Minimum Requirements

Latest version of Valheim.

This server is not vanilla conpatible. Please use the modpack (install instructions below).

Playing Valheim With Mods

Fear not, for it is not hard with the right launcher.

There are a few great launchers that make it easy to add mods, such as r2modman, which is what we use. Simply install the launcher, then follow the instructions as seen below:
- Open R2modman
- Select Valheim (Pressing star brings it to the front when you open the mod manager)
- Select Steam
- Select "Import" to import a profile
- Select "Import new profile" then "From code"
- Type the code 01906ba6-c795-a9ef-4c01-99c97ec1c03e into the text box
- Create the profile
- Wait for the mods to download
- Press "Start Modded" to launch the game

Modpack code: 01906ba6-c795-a9ef-4c01-99c97ec1c03e

If you need help with setting up the modded launcher, or want to install manually, come ask in the Discord.

FAQ

Q: Are there any world modifiers?

A: We're playing in immersive mode, so no portals and no map aside from a mod that enables it in the cartography table (this feature should be in the base game imo). Combat is set to hard but the death penalty is reduced.

Q: Can I have the world seed?

A: No. We're playing Immersive mode, so that kind of defeats the point.

Q: What mods are installed on the server?

A: Here's a list I made, just for you:

  • Better_Cartography_Table by nbusseneau

    • Allows precise control over sharing pins and map exploration via cartography tables. Supports private pins, public pins, and guild pins. Supports NoMap runs, giving cartography tables a purpose in NoMap.

  • PlantEverything by Advize

    • Allows your cultivator to plant berry bushes, thistle, dandelions, mushrooms, previously unavailable tree types, and other decorative flora.

  • PlantEasily by Advize

    • Allows you to plant evenly spaced crops, either one at a time or in any amount of rows and columns, with grid snapping features and invalid crop placement prevention. Harvest in bulk and optionally auto replant crops.

  • Gizmo by ComfyMods

    • Rotate along all 3 axis, and with customizable rotation intervals.

  • Seasons by shudnal

    • 4 Seasons, effecting crop yield and growth, among other things.

  • AdvancedTerrainModifiers by Searica

    • Quality of life building mod that improves how terrain manipulation with the hoe and cultivator works and adds new terrain manipulation tools.

  • EquipmentAndQuickSlots by RandyKnapp

    • Give equipped items their own dedicated inventory slots, plus three more hotkeyable quick slots.

  • PotteryBarn by ComfyMods

    • Public build extension that adds existing game prefabs to the Hammer.

  • AzuAreaRepair by Azumatt

    • Repair pieces in a specified area around the player when using the hammer to repair.

  • UseEquipmentInWater by LVH-IT

    • Allows you to use your Equipment while swimming in water

  • BiggerBoatMapExploreRadius by nearbear

    • Increases the exploration radius while in a boat.

  • Heightmap_Unlimited_Remake by Digitalroot

    • Allows players to raise or lower terrain as much as they want!

  • SpeedyPaths by Nextek

    • Gives more purpose to paths. Increases player speed on paths and constructions. Reduces stamina consumed while sprinting on paths.

  • EpicLoot by RandyKnapp

    • This mod aims to add a loot drop experience to Valheim similar to Diablo or other RPGs. Monsters and chests can now drop Magic, Rare, Epic, Legendary, or Mythic magic items. Each magic item has a number of magic effects on it, that give bonuses to the item or your character when that magic item is equipped.

  • BoatAdditions by blacks7ar

    • Adds 3 small and 5 new large boats with anchors and lamps and a boatyard that can help you on your journey. Each boat has its own anchors, if you logout while boats are still anchored, if you log back in they will still be anchored, and adds anchor function to vanilla boats. Each boat has its own lamp that can help you navigate at night. Adds a new crafting station called Boatyard, you can use this new crafting station to craft parts of boats which then you can use to build your boats.

  • HiddenDoors by Frogger

    • This mod just adds doors that look exactly like the walls.

  • CustomLoadingScreens by cjayride

    Customized loading screens and tip screens.

Now go play Valheim.
